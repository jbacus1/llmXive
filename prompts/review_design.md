# Technical Design Review Prompt

Please conduct a comprehensive review of the following technical design document:

**Document Content:**
{content}

## Review Criteria
Evaluate the document on:
1. Technical feasibility (1-10 scale)
2. Novelty and innovation (1-10 scale)
3. Clarity and completeness (1-10 scale)
4. Methodology soundness (1-10 scale)
5. Resource requirements realism (1-10 scale)

## Review Structure Required
**SCORE: [overall score 1-10]**

**DETAILED FEEDBACK:**
- Strengths: [list key strengths]
- Weaknesses: [list major weaknesses]
- Missing elements: [what's missing]
- Recommendations: [specific improvements]

**VERDICT:** [ACCEPT/REVISE/REJECT]

## Variables
- {content}: The technical design document content to review
