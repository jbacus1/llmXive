# Research Decisions: Evolutionary Pressure on Alternative Splicing

**Date**: 2024-01-15  
**Context**: Paper Stage Planning for PROJ-002

## Decisions & Rationale

### 1. Target Venue & LaTeX Class
**Decision**: `arxiv-style` (Preprint)  
**Rationale**: Maximizes reproducibility access and allows immediate validation of pipeline checksums before peer review. Aligns with Constitution Principle V (Reproducibility) by using a standard, widely supported class.  
**Alternative Rejected**: `acmart` (Conference) requires stricter page limits that may compromise Methods transparency. `Nature` requires specific formatting that may obscure raw data links.

### 2. Figure Generation Toolkit
**Decision**: `matplotlib==3.7.0` + `seaborn==0.12.2`  
**Rationale**: Python ecosystem aligns with the analysis pipeline (STAR/rMATS output processing). Ensures figures are generated from real data in `data/` per Constitution Principle II.  
**Alternative Rejected**: `ggplot2` (R) would require a separate R environment, increasing build complexity. `Plotly` introduces interactivity dependencies not supported in static PDFs.

### 3. Citation Manager & Reference Style
**Decision**: `biblatex` with `Nature` style  
**Rationale**: Standard for biology journals. `biblatex` allows flexible verification hooks.  
**Constraint**: Citations MUST come from `state/citations/PROJ-002-evolutionary-pressure-on-alternative-spl.yaml` where `verification_status=verified`. No new citations invented.  
**Alternative Rejected**: EndNote (proprietary), BibTeX (legacy, less flexible for verification metadata).

### 4. Comparative Species Framework
**Decision**: Human (Homo sapiens) vs. Chimpanzee (Pan troglodytes) + Mouse (Mus musculus) outgroup  
**Rationale**: Constitution Title specifies "Primates". Human/Chimp is the primary evolutionary pressure comparison. Mouse serves as mammalian outgroup for conservation context.  
**Clarification**: Exact species pairs pending confirmation from research-plan artifacts (see Spec Assumptions). Plan assumes standard primate comparative genomics set.  
**Alternative Rejected**: Single-species analysis (fails comparative requirement).

### 5. Splicing Event Types
**Decision**: Exon Skipping (SE), Mutually Exclusive Exons (MXE), Alternative 5'/3' Splice Sites (A5SS/A3SS)  
**Rationale**: Standard rMATS event types. Spec mentions these in User Story 4 acceptance scenarios.  
**Constraint**: Retained Intron (RI) excluded if low signal-to-noise ratio in research artifacts.  
**Alternative Rejected**: Custom event definitions (violates rMATS reproducibility).

## Verification Dependencies

- **Citations**: Must resolve `verification_status=verified` from Research Stage.
- **Data**: All figures must trace to `data/` CSV/JSON files generated by `scripts/`.
- **Build**: `pdflatex` must return 0 on `paper/source/main.tex`.
