# Quickstart: Mindfulness and Social Skills Study

## Overview

This quickstart guide helps researchers set up the data collection and analysis pipeline for the mindfulness intervention study.

## Prerequisites

### Software Requirements

| Tool | Version | Installation |
|------|---------|--------------|
| Python | 3.11+ | `pyenv install 3.11.7` |
| pip | 23.0+ | `python -m pip install --upgrade pip` |
| git | 2.30+ | System package manager |
| R | 4.3+ | System package manager (optional, for advanced stats) |

### Dependencies

Create a virtual environment and install dependencies:

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install from requirements.txt (to be generated by Implementer)
pip install -r requirements.txt
```

**Expected requirements.txt includes**:
- pandas>=2.0.0
- numpy>=1.24.0
- scipy>=1.11.0
- pingouin>=0.5.3
- pydantic>=2.0.0
- jsonschema>=4.18.0
- pytest>=7.4.0
- matplotlib>=3.7.0
- seaborn>=0.12.0

## Data Collection

### Starting a New Participant

```bash
python -m src.cli.cli register-participant \
    --age 10 \
    --sex "Male" \
    --diagnosis-date "2024-01-15" \
    --diagnosis-facility "ABC Clinic"
```

### Recording an Assessment

```bash
python -m src.cli.cli record-assessment \
    --participant-id "ASD-0001" \
    --timepoint "baseline" \
    --srs-total 65.0 \
    --ssis-total 85.0 \
    --assessor-id "ASS-001"
```

### Recording an Intervention Session

```bash
python -m src.cli.cli record-session \
    --participant-id "ASD-0001" \
    --session-number 1 \
    --duration 32 \
    --attendance "present" \
    --homework-completed true
```

## Data Validation

### Validate Raw Data Against Schema

```bash
python -m src.cli.cli validate-data \
    --input data/raw/assessments.csv \
    --schema contracts/assessment.schema.yaml
```

### Check Data Completeness

```bash
python -m src.cli.cli check-completeness \
    --data data/validated/ \
    --report data/reports/completeness.html
```

## Analysis Pipeline

### Run Descriptive Statistics

```bash
python -m src.cli.cli analyze-descriptive \
    --data data/processed/clean_data.csv \
    --output data/results/descriptive.json
```

### Run Primary Analysis (ANOVA)

```bash
python -m src.cli.cli analyze-anova \
    --data data/processed/clean_data.csv \
    --dependent "srs_total" \
    --between "group_assignment" \
    --within "timepoint" \
    --output data/results/anova_results.json
```

### Generate Report

```bash
python -m src.cli.cli generate-report \
    --results data/results/ \
    --template docs/report_template.md \
    --output docs/final_report.md
```

## Testing

### Run Full Test Suite

```bash
pytest tests/ -v --tb=short
```

### Run Contract Tests Only

```bash

```

### Run Tests with Coverage

```bash
pytest tests/ -v --cov=src --cov-report=html
```

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Schema validation fails | Check `contracts/` for latest schema; ensure data matches format |
| Missing dependencies | Run `pip install -r requirements.txt` |
| Permission errors | Check file permissions on `data/` directories |
| IRB data access denied | Contact IRB administrator for access credentials |

### Getting Help

1. Check `docs/` for detailed documentation
2. Review `tests/` for usage examples
3. Contact project maintainers via GitHub Issues
